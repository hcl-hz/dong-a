# 동아대학교 학교소개 페이지 — 디자인 가이드

> 대상 파일: `index copy.html` (학교소개 하위 페이지) · 스타일: `assets/css/style.css`

## 색상 팔레트

| 역할 | 이름 | HEX | 용도(제안) |
|------|------|-----|-----------|
| 주조색 | Primary | `#5409DA` | 핵심 브랜드 컬러 — 주요 버튼, 강조 헤딩, 포인트 요소 |
| 보조색 | Secondary | `#4E71FF` | 보조 강조 — 링크, 보조 버튼, 아이콘, 카드 배경 |
| 강조색1 | Accent 1 | `#8DD8FF` | 밝은 강조 — 배지, 아이콘 배경, 도형/장식 |
| 강조색2 | Accent 2 | `#BBFBFF` | 가장 옅은 강조 — 섹션 배경 틴트, 호버 배경, 그라데이션 끝색 |

```
주조색 #5409DA   ████  딥 퍼플
보조색 #4E71FF   ████  블루
강조색1 #8DD8FF  ████  라이트 스카이
강조색2 #BBFBFF  ████  페일 시안
```

### 사용 원칙(초안)
- 텍스트/주요 액션: **주조색**
- 면적이 큰 배경·카드: **보조색** 또는 **강조색** (가독성 위해 옅게)
- 그라데이션: 주조색 → 보조색, 또는 강조색1 → 강조색2
- 본문 텍스트는 팔레트 색 대신 중성 다크(가독성)

## 섹션 구성

1. **메인 (히어로)** — 풀스크린 이미지 슬라이드 + 카피 + 좌하단 핵심 바로가기 칩(QUICK · 입학안내 · 장학 · 캠퍼스맵)
2. **동아뉴스** — 뉴스 슬라이더(좌 텍스트 + 우 확장형 이미지), 좌상단 장식 도형
3. **대학공지** — 2열(동아 캘린더 + 공지사항 리스트)
4. **주요서비스** — 좌측 탭(공통/예비동아인/재학생) + 우측 아이콘 캐러셀, 우상단 장식 도형
5. **동아파워** — 흰→블루 그라데이션 배경 + 카드 마퀴(통계/강점), 좌우 화살표
6. **푸터** — 정책 링크 · 캠퍼스 주소 · 셀렉트 메뉴 (다크 네이비)

- **우측 플로팅 도크**: 예비동아인 · 재학생 · 교직원 · 학부모 · TOP

> 섹션 구성·세부는 추가 지침에 따라 보강.
